Event webhook

Event Webhooks bieten einen Weg, über Änderungen benachrichtigt zu werden.

Die Nutzung dieser Funktion ohne Programmierkenntnisse kann anspruchsvoller sein. Wenn Du Fragen hast, melde Dich gerne bei uns.

Aktuell gibt es das Event form.submitted. Dieses Event kann per Event Webhook Workflow Action gesendet werden.

Das Event sendet Daten der gesendeten Antwort an den angegebenen Endpunkt.

Formenio sendet ein Secret als Header mit: formenio: workflow-id

Du siehst die Workflow ID nach speichern des Workflows. Stelle so sicher, dass die Anfrage wirklich von Formenio stammt.

Hier ist ein Beispiel Event, das gesendet werden könnte:


JSON
{
   "event":"form.submitted",
   "formId":"69f1f9359b009a4048d629e4",
   "responseId":"2ed39c31-a9f9-4c34-9120-3a35f3a09415",
   "fields":{
      "harswmkul":{
         "label":"Textfeld Beispiel",
         "type":"text",
         "value":"Eingabetext"
      },
      "rn1juqvwg":{
         "label":"Kartenauswahl",
         "type":"cardSelect",
         "value":"Option 2"
      },
      "c7o9wpmuk":{
         "label":"Dateifeld Beispiel",
         "type":"file",
         "value":[
            "testfile2.txt"
         ]
      }
   },
   "status":null,
   "submittedAt":"2026-04-29T12:30:09.268Z",
   "timestamp":"2026-04-29T12:30:09.273Z"
}


Weitere Events sind geplant, jedoch noch nicht umgesetzt.

Bei Fragen zu Webhooks, melde Dich gerne direkt bei uns: support@formenio.de
