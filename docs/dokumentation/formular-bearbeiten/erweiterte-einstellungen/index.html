Erweiterte Einstellungen

Hier sind alle Formulareinstellungen, die Du bearbeiten kannst, aufgelistet.
Öffne "Einstellungen" oder "Style" über die Formularbearbeitung.

Einstellungen

Einstellung

Beschreibung

Öffentlich

Steuert, ob Dein Formular abrufbar ist. Falls nicht, kann dein Formular ausschließlich von Dir über die Vorschau angesehen werden.

Primäre Domain

Über diese Domain kann das Formular abgerufen werden.
Das Formular ist immer über app.formenio.de erreichbar.
Du kannst hier eine zusätzliche eigene Subdomain (mehr erfahren) setzen, z.B. meine-firma.formenio.app.

Fortschritt anzeigen

Wenn aktiv, wird den NutzerInnen ein Fortschrittsbalken angezeigt, je nachdem, auf welcher Seite sie sich gerade befinden. Bei einseitigen Formularen wird kein Balken angezeigt.

Analytics einschalten

Wenn aktiv, werden Analytics-Daten der NutzerInnen gespeichert (mehr erfahren). Wenn nicht aktiv, sind nicht alle Funktionen bezüglich Analytics verwendbar. Wichtig: Beim deaktivieren von analytics werden keine Daten gelöscht.

Optional: Formular aufrufbar ab

Stelle hier ein Datum (und optional Uhrzeit) ein, ab dem dein Formular online aufrufbar sein soll. Wird das Formular vorher aufgerufen, wird angezeigt, dass das Formular noch geschlossen ist.

Optional: Formular absendbar bis

Stelle hier ein Datum (und optional Uhrzeit) ein, bis zu dem das Formular Antworten entgegennehmen soll. Wird nach diesem Zeitpunkt eine Antwort gesendet, wird diese nicht gespeichert und eine Fehlermeldung wird zurückgegeben. Besucht ein/e NutzerIn das Formular nach diesem Zeitpunkt, wird angezeigt, dass das Formular aktuell geschlossen ist.

Absenden Text

Dieser Text wird als Button angezeigt, mit dem NutzerInnen das Formular absenden können. Standard: "Absenden"
Bei Kontaktanfragen kann beispielsweise "Anfrage senden" angegeben werden, bei Umfragen zum Beispiel "Teilnehmen".

Socials: Titel

Mit dieser Einstellung stellst Du ein, welche Vorschau des Formulars auf Social Media angezeigt wird. Diese Einstellung beeinflusst den Titel, den NutzerInnen sehen.

Socials: Beschreibung

Mit dieser Einstellung stellst Du ein, welche Vorschau des Formulars auf Social Media angezeigt wird. Diese Einstellung beeinflusst die Beschreibung, die NutzerInnen sehen.

Formenio Logo ausblenden

Stelle hier ein, ob das Logo von Formenio auf deinem Formular ausgeblendet werden soll.

Style

Einstellung

Beschreibung

Hintergrundfarbe

Stellt die Hintergrundfarbe des Formulars ein. Achte auf eine saubere Lesbarkeit der Texte und auf eine gute Erkennbarkeit der Felder.

Kann hilfreich sein, um beispielsweise bei eingebetteten Formularen diese der Webseite anzupassen. Muss im Normalfall nicht geändert werden.

Buttonfarbe

Dies ist die Farbe der Buttons deines Formulars sowie des Fortschrittbalkens. Stelle hier Deine gewünschte Akzentfarbe ein. Diese Einstellung wird am häufigsten genutzt, um Formulare mit einer repräsentativen Farbe anzupassen.

Textfarbe

Du kannst hier die Textfarbe anpassen, um Texte sauber und leicht lesbar zu halten. Nicht alle Texte werden mit dieser Einstellung beeinflusst.

Theme (experimentell)

Hierbei handelt es sich um eine Vorabeinstellung, die sich aktiv in Entwicklung befindet.
Hiermit kannst Du das generelle Aussehen des Formulars anpassen.
Empfehlung: Ändere das Theme nur, wenn das Formular noch unveröffentlicht ist, und schaue Dir die Änderungen über die Vorschau an.

Schriftgröße

Ändere die Schriftgröße, um das Formular an Inhalt und Zielgruppe anzupassen.
Hinweis: Noch nicht alle Texte werden hiervon beeinflusst.

Breite

Dein Formular ist standardmäßig auf eine Breite von 45% der Bildschirm- bzw. Fenstergröße eingestellt.
Dies kannst Du hier ändern.
Hinweis: Bei mobilen Endgeräten sowie als eingebettetes Formular ist die Breite immer 100%.

Horizontale Ausrichtung

Stelle hier ein, wie sich das Formular nach links und rechts auf der Seite ausrichten soll.
Ist die Breite auf 100% gestellt, beeinflusst diese Einstellung die Darstellung nicht.

Vertikale Ausrichtung

Stelle hier ein, wie sich das Formular nach oben und unten auf der Seite ausrichten soll.
Hast Du mehrere Felder auf einer Seite, oder beispielsweise viele Auswahlkarten, kann die Formulargröße auch die Höhe des Fensters überschreiten. Dann ändert diese Einstellung nichts.

Ecken-Radius

Hiermit kannst du steuern, wie abgerundet Ecken von Buttons und EIngabefeldern dargestellt werden sollen.
Bei manchen Themes ändert dies nichts mehr. Standardmäßig bei dem Theme "Light" kannst du hiermit jedoch ganz einfach die Ecken steuern.
